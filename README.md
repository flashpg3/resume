## 页面结构

### 1. 顶部工具栏（64px）

- **左侧 Logo**：「简历助手」Bold 20px
- **右侧操作区**：
  - 「清空」按钮 — outlined，灰色文字
  - 「导出 PDF」按钮 — 实心深色主按钮

### 2. 左侧信息输入区（480px，可滚动）

五大模块，自上而下排列：

| #    | 模块     | 字段                        | 交互                                     |
| ---- | -------- | --------------------------- | ---------------------------------------- |
| 1    | 基本信息 | 姓名、电话、邮箱            | 3个输入框                                |
| 2    | 教育背景 | 学校、专业、开始/结束日期   | `[+ 添加]` 增加条目，`[删除]` 移除       |
| 3    | 工作经历 | 公司、职位、开始/结束日期   | `[+ 添加]` + `[删除]`，含 `✨AI助写` 提示 |
| 4    | 项目经验 | 项目名、角色、开始/结束日期 | `[+ 添加]` + `[删除]`，含 `✨AI助写` 提示 |
| 5    | 专业技能 | 技能标签 + 输入框           | 标签展示（Python/JS/React）+ 输入添加    |

### 3. 右侧实时预览区

- A4比例（595×842px）白底黑字预览卡片
- 实时展示简历结构：姓名 → 联系方式 → 教育背景 → 工作经历 → 项目经验 → 专业技能
- 含示例填充内容（张明远 · 北京大学 · 字节跳动实习等）

### 4. 空状态引导

- 底部提示：「从上方模块开始填写，右侧实时预览简历效果」

## 交互原型流程

```
[打开应用] → 空状态引导
     ↓
填写左侧表单模块 → 右侧实时同步预览（200ms内）
     ↓
点击[+添加] → 新增空白条目
点击[删除] → 移除条目 + 预览同步
点击[✨AI助写] → 预留交互入口（P1）
     ↓
点击[导出PDF] → 生成A4 PDF下载
点击[清空] → 二次确认后重置所有数据
```
